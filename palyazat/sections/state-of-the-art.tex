\noindent
Recent advances in XR technologies and digital twin systems have opened new possibilities in cultural heritage preservation. However, current applications are often limited to isolated case studies or indoor environments.

This project extends the state of the art by:
\begin{itemize}
    \item Applying XR technologies in complex outdoor archaeological contexts
    \item Integrating large-scale, multi-region datasets
    \item Combining scientific documentation with public accessibility
\end{itemize}

It builds on existing work in digital archaeology, photogrammetry, and immersive technologies, while introducing scalable and standardized workflows.
