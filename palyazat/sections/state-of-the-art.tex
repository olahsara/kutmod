Recent advances in XR technologies and digital twin systems have opened new possibilities in cultural heritage preservation. However, existing solutions are often limited to isolated case studies or indoor environments.

\medskip
Current limitations include:
\begin{itemize}
    \item Lack of scalability
    \item Limited outdoor applicability
    \item Weak integration between scientific documentation and public accessibility
\end{itemize}

The project advances the state of the art by:
\begin{itemize}
    \item Applying XR technologies in complex outdoor archaeological contexts
    \item Integrating large-scale, multi-region datasets
    \item Combining scientific documentation with immersive public access
\end{itemize}
