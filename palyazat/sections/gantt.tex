The project timeline spans 24 months and is structured through a phased implementation of Work Packages.

Key milestones include:
\begin{itemize}
    \item Completion of data collection (Month 6)
    \item XR platform prototype (Month 12)
    \item Pilot testing (Month 18–22)
    \item Final dissemination event (Month 24)
\end{itemize}
